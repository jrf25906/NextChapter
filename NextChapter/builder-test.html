How to Use

  1. Click "Go to Content Dashboard" above
  2. Click on the "Home" page to edit it
  3. In the visual editor, look for the components panel on the left
  4. Drag and drop components onto the canvas
  5. Click "Publish" when done
  6. Check your React Native app to see the changes

Troubleshooting

If you don't see your custom components:

  • Make sure the dev server is running (check terminal)
  • Try refreshing the Builder.io page
  • Look for a "Blocks" or "Components" tab in the left sidebar
  • Check the console for any errors (F12)
